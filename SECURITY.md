# Security Policy

## Reporting a Vulnerability

Email security@mctx.ai with details. We will acknowledge within 48 hours and aim to resolve issues within 30 days.

Do not file public issues for security concerns.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| latest  | Yes       |

## AI-Generated Content

The sampling feature allows servers to request content from connected AI clients. Developers are responsible for validating content received through this feature.
